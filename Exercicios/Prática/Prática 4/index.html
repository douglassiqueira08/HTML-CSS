Negrito e itálico

Aqui temos um termo em negrito utilizando a tag strong.

Aqui temos um termo em itálico utilizando a tag em.

Texto delatado

Aqui temos um texto que pode ser lido, mas deve ser deconsiderado, utilizando a tag DEL.

Texto inserido

Ao contrário do texto deletado, o texto inserido indica que deve ser lido, utilizando a tag ins.

Texto sobrescrito

Para digitar x2 + 3 por exemplo. Para o 2 ficar em cima, digitamos a tag SUP.

Texto subscrito

Quando precisando usar a letra ou número um pouco mais abaixo. Exemplo: H2O. Uitilizamos a tag SUB.

Citações

Quando usando citações em textos sem fazer deslocação, podemos utilizar tag Q.

Também podemos usar citações mais logas, um paragráfo próprio por exemplo , nesse caso usando a tag blockquote.

Exemplo:

Lorem ipsum dolor sit amet consectetur adipisicing elit. Voluptatem earum, nihil, temporibus natus alias commodi repellendus voluptatum reiciendis nobis veniam mollitia a assumenda. Nostrum inventore voluptatibus vero animi minima. Quasi?
